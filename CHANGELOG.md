# Changelog

## 0.1.0

- Initial public repo package as `YT-TranscriptClick`.
- Adds a floating YouTube subtitle extraction panel for Tampermonkey and compatible userscript managers.
- Captures real YouTube subtitle network responses when direct `timedtext` requests return empty responses.
- Supports copying extracted subtitle text to clipboard and downloading a `.txt` file.
- Adds GitHub `downloadURL` and `updateURL` metadata for userscript manager updates.
